### InterNet基础知识
- InterNet简介
```text
在了解什么是因特网之前，首先先要了解网络是什么，所谓因特网就是网络的网络，那么问题来了网络的定义又是什么呢？
网络的定义很简单 网络就是一组计算机设备或者其他设备互相连接起来组合这个就叫做网络，那么现在因特网的定义就比较清晰，
可以简单的理解为全世界所有网络设备的总和(不太准确)但是目前可以这样简单的理解
```

- InterNet如何工作
```text
从大体上来看，因特网使用一系列标准化协议(standard protocols) 来连接计算机和其他的网络设备，这些协议定义了信息如何在不同设备之间的方法
从而保证信息交换的可靠传输和安全性，互联网的核心是互联路由器，路由器可以看做特殊的计算机设备，路由器可以连接两个不同的异构网络，当你发送数据
到互联网的时候，数据会被切片为更小的packet到路由器(router),路由器会检查这个得到的数据包 并将它发送到目的路径上的下一个路由器，这个过程
将会持续进行至到data到达目的地址

为了保证目的主机收到的数据是正确的，internet有不很多不用的协议来验证这个数据是否是正确的，常见的有Tcp(Transmission Control Protocol)
协议和IP(Internet Protocol)协议 IP协议负责将数据包路由到其正确的目的地，而TCP确保数据包可靠地传输并按正确顺序传输。简单来讲Tcp/IP协议
就是保证数据正确并且有效的传输到目的主机

除此之外还有很多其他的协议也确保数据在internet上的交换 包括DNS(Domain Name System),HTTP(Hyper Text Transmit Protocol),作为开发者
需要对这些协议有一个稳固的理解 来了解他们如何共同控制数据在因特网安全有序的交换

```

- 基本概念和术语
```text
Packet 通过网络传送的一小包信息
Router 将信息在不同网络间传送的设备
IP Address IP地址 i.e 192.168.77.77
Domain Name 域名 i.e www.google.com
DNS 域名解析系统 将域名解析为IP地址
HTTP 超文本传输协议 用于在客户端和服务端之间传输数据
HTTPS 加强安全版的HTTP协议
SSL 用于确保在互联网上传输的数据的安全性和完整性的协议(未接触)
```

- 协议在互联网中的作用
```text
协议在互联网中扮演及其重要的作用 从上面可以看出信息数据传递都离不开协议的作用 上面已经介绍了基本的几个协议这里补充一个UDP协议 UDP协议也是
网络协议中的一种这里只简单说明与TCP协议的区别 首先UDP协议不需要建立连接 其次UDP协议并不可靠也就是说数据包可能会丢失或损坏，但传输速度更快
最后就是UDP适用于对实时性要求更高的网络而网络传输的信息并不是它关注的重点
```
